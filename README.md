# Flutter-ShashaShany-GetYourIdea

Get Your Idea
Deskripsi Aplikasi
Motion Idea Tracker adalah aplikasi mobile berbasis Flutter yang dirancang untuk membantu pengguna mengumpulkan dan menyimpan ide-ide atau artikel pendek dari internet. Aplikasi ini mengintegrasikan data dari API publik (JSONPlaceholder) sebagai sumber inspirasi, yang kemudian dapat disimpan oleh pengguna ke dalam koleksi pribadi menggunakan local storage (SQLite). 

Fitur Utama
Public Ideas Explorer (Read API): Menampilkan daftar ide/artikel yang ditarik secara real-time dari eksternal API.
Save to Collection (Create): Pengguna dapat menyimpan ide yang menarik ke dalam local database SQLite.
My Saved Ideas (Read Local): Menampilkan daftar seluruh ide yang telah disimpan pengguna.
Edit Content (Update): Pengguna dapat memodifikasi teks/ide yang sudah disimpan sesuai dengan preferensi pribadi.
Remove Idea (Delete):Fitur untuk menghapus ide dari database lokal.
State Management: Diimplementasikan menggunakan GetX.
